# How to run

<ol>
<li>
Clone the app:

```
git clone https://github.com/daniel-souza01/be-the-hero_backend.git
```

</li>

<li>
Install all dependencies:

```
yarn
or
npm install
```

</li>

<li>

Copy `.env.example` to `.env` .

</li>

<li>
Create and start the application and postgres containers:

```
docker-compose up
```

</li>
</ol>
